```
git clone https://github.com/sp3crtre/Buggy_Web.git
```

#### Make it executable and grant root access

```
sudo chmod +x BuggyWeb.sh
```

#### Now, run it by simply typing the command below:

```
sudo ./BuggyWeb.sh
```
